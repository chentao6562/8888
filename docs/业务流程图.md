## 二、模块业务流程

### 2.1 项目管理流程

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│  创建项目 │ ──→ │ 设定目标  │ ──→ │ 分配人员  │ ──→ │ 制定指标  │ ──→ │ 跟踪执行  │
└──────────┘     └──────────┘     └──────────┘     └──────────┘     └──────────┘
                                                                          │
     ┌────────────────────────────────────────────────────────────────────┘
     ↓
┌──────────┐     ┌──────────┐     ┌──────────┐
│  定期复盘 │ ──→ │ AI总结   │ ──→ │ 优化改进  │
└──────────┘     └──────────┘     └──────────┘
```

**详细步骤：**
1. **创建项目**：填写项目名称、背景、核心目标
2. **设定目标**：设定可量化的项目目标（如：月线索量 500）
3. **分配人员**：为项目分配团队成员和角色
4. **制定指标**：为每个岗位设定 SMART 指标
5. **跟踪执行**：实时监控指标完成情况
6. **定期复盘**：周/月度复盘会议
7. **AI 总结**：AI 自动生成复盘报告和建议
8. **优化改进**：根据复盘结果调整策略
