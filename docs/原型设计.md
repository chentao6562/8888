## 十、核心交互说明

### 项目流程节点
- 每个项目有独立的流程（可自定义阶段）
- 点击阶段可展开查看该阶段任务
- 任务可拖拽调整顺序和阶段
- 完成所有任务后可推进到下一阶段

### 协作功能
- **讨论**: @提及成员，支持回复
- **待办**: 可分配给指定成员，设置截止日期
- **日程**: 显示项目相关的会议和节点
- **文件**: 按文件夹分类管理项目文件

### 经验沉淀
- 复盘时可选择"存入经验库"
- 经验按分类组织（内容创作/线索转化/项目管理等）
- 新建项目时智能推荐相关经验
- AI 根据当前项目阶段推荐经验

### 数据流转
```
外部平台数据 → 导入系统 → 关联到具体项目 → 生成看板 → AI分析 → 输出建议 → 沉淀经验
```

## 十九、新增功能交互说明

### 官网内容管理
- 后台编辑 Banner、服务介绍、案例等内容
- 实时预览，保存后自动同步到前端官网
- 支持图片上传、富文本编辑

### 项目预警系统
- 阶段超时自动触发预警
- 指标异常自动监控
- 多级预警（黄色→红色→紧急）
- 多渠道通知（站内、邮件、微信）

### 领导驾驶舱
- 仅管理层角色可见
- AI 自动生成每日/每周分析报告
- 一键查看预警项目
- 团队绩效可视化

### 项目模板
- 预设流程阶段和任务
- 新建项目时可选择模板
- 支持自定义模板

### 项目评论 & 匿名吐槽
- 项目阶段内支持评论反馈
- 可选择匿名发送
- 全局吐槽墙 100% 匿名
- 管理员可回复但不能删除

### 工作汇报
- 日报/周报/月报三种类型
- 可关联项目任务自动填充
- AI 辅助生成汇报
- 管理者可查看团队汇报

### 企业文化
- 公告发布与阅读追踪
- 文化价值观展示
- 学习资料推送
- 荣誉墙表彰
